# 2026-05-25 開発ミーティングgit hub 勉強会 議事録

## 実施内容

山崎部長主催にて、GitHub活用に関する勉強会を実施。

現状の各メンバーのGitHub活用状況や理解度について、
各自より進捗・実施内容の共有を行った。

今後については、
単なる説明会形式ではなく、

「各自が出来るようになったこと」
「実際に試した内容」
「業務へどう活用したか」

を定期的にお披露目しながら、
部内メンバー間で切磋琢磨していく運用へ
切り替えていく方針となった。

---

## GitHub / VS Code 運用例共有

実際の運用例として、
VS Code を活用した GitHub 操作について実演を実施。

### 実施内容

- GitHubリポジトリのクローン
- フォルダ作成
- mdファイル作成
- 画像フォルダ管理
- Git操作（add / commit / push）
- ドキュメント管理運用イメージ共有

### 確認できた点

- VS Code上で一通りの操作が完結可能
- md形式により技術情報の蓄積・共有が容易
- GitHub上で履歴管理できるメリットが大きい
- 個人管理ではなく、チーム共有資産として活用可能

---

## 今後の進め方

- 小さくても実際に触ることを優先
- 各自が試した内容を都度共有
- 技術情報をGitHubへ蓄積
- 「分かる人だけが使う」状態にしない
- 日常業務へ徐々に組み込んでいく
